# 가상클래스란 무엇일까?

가상클래스(Pseudo-Class)는 선택될 요소(element)의 특별한 상태를 지정하는 선택자(selector)에 추가된 키워드 입니다.

```html
<div>
  <button class="btn">버튼입니다</button>
</div>

<style>
  .btn {
    font-weight: bold;
  }
</style>
```
